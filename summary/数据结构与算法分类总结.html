连续子数组问题

连续子数组问题均可用前缀和解决. 若用P[i]表示[0, i]项的和, 则[i, j]项的和为P[j] - P[i-1], 解题的关键是寻找等式P[j] ⊙ A = P[i-1] ⊙ B(其中⊙表示任意运算符, A, B为任意常数). 此外, 注意需要添加边界条件, 前0项的和为0.

树状结构

二叉树/二叉搜索树

二叉树中最基本的算法是其四种遍历方法, 包括:

  • 层次遍历, 属于BFS;
  • 前序遍历, 根-左-右, 题目中可能需使用其镜像顺序, 即根-右-左, 属于DFS;
  • 中序遍历, 左-根-右, 题目中可能需使用其镜像顺序, 即右-根-左, 属于DFS;
  • 后续遍历, 左-右-根, 题目中可能需使用其镜像顺序, 即右-左-根, 属于DFS.

几乎所有与二叉树相关的问题都能用特定遍历方法的变形来解决. 二叉搜索树的中序遍历是有序排列.

图状结构

DFS

图的DFS其实是回溯算法的一个特例, 存在以下几点区别:

  • 遍历对象: 回溯算法遍历的是一棵多叉树, 而图的DFS遍历的是一个图. 但是本质上是一样的, 无环图就是一个多叉树, 对于有环图在终止条件出增加一个判断, 如果结点已经被访问过即结束即可.
  • 遍历目的: 回溯算法的遍历目的是得到从根结点到叶节点的路径(如排列问题)或从根节点到某一内部结点的路径(如子集问题). 而DFS的目的是依次访问图中的每个结点.
  • 遍历数量: 一般来说回溯算法如果需要得到全部结果则需要把遍历整棵多叉树, 但只需要获得一个结果则不需要. 但是图的DFS一般会遍历整个图.

BFS

BFS主要用于寻找最短路径. BFS相对DFS的最主要的区别是: BFS找到的路径一定是最短的, 但代价就是空间复杂度比DFS大很多.

回溯算法

回溯算法实际上是一棵多叉树的遍历问题, 每个叶节点都是问题的一个解, 如果问题只需要求得一个解, 那么可将回溯函数的返回值定义为boolean, 一旦求得一个解即返回true, 在遍历过程中可采用适当的方法剪枝. 回溯法的三要素为:

  • 回溯路径;
  • 选择列表;
  • 结束方法;

平方复杂度

  • 300. 最长上升子序列** ($$dp[i]$$表示以nums[i]结尾的数组成的最长上升子序列长度, $$ dp[i] = \left{\begin{matrix} max(dp[j]) + 1; 0 \leqslant j < i \ 1; i = 0 \end{matrix}\right. $$, 本题存在时间复杂度为$$ O(nlogn) $$的解法)
  • 322. 零钱兑换** ($$ dp[i] $$表示兑换金额i使用的最少硬币数量, $$ dp[i] = \left{\begin{matrix} min(dp[i-coins[j]]) + 1; 0 \leqslant j < \left |coins \right |, i \geqslant coins[j], dp[i-cons[j]] \ avaliable\ 0; i = 0 \end{matrix}\right. $$)
